# TEAM: Techno-Economic Assessment and Manipulation framework

TEAM stands for **Techno-Economic Assessment and Manipulation** framework. It is pronounced the same way as the English word [team](https://dictionary.cambridge.org/pronunciation/english/team) /tiːm/.

TEAM is a framework for performing techno-economic assessments in the context of energy and climate-mitigation studies.

For more help, please consult the [documentation](https://philippverpoort.github.io/team/latest).

## Installation

You can install TEAM via `pip` directly from GitHub:

```bash
pip install git+https://github.com/PhilippVerpoort/team.git
````

Or clone and install locally:

```bash
git clone https://github.com/PhilippVerpoort/team.git
cd team
pip install .
```

> **Note:** The package will be published on PyPI in the near future for simpler installation.
